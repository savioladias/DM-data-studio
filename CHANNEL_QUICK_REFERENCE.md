## 60-Second Setup Guide

### For Google Services (GA4, GSC, Google Ads):
1. Click **Connect** on the channel
2. Sign in with Google
3. Pick the property/website/account
4. Done ✓

### For Meta Services (Facebook, Instagram):
1. Click **Connect** on the channel
2. Sign in with Facebook
3. Select your page/account
4. Done ✓

### For LinkedIn:
1. Click **Connect**
2. Sign in with LinkedIn
3. Select company page
4. Done ✓

### For YouTube:
1. Click **Connect**
2. Sign in with Google
3. Select your channel
4. Done ✓

## Permission Checklist

Before connecting, make sure you have:

**Google Analytics 4**
- [ ] Access to the GA4 property
- [ ] Admin or Editor role

**Google Search Console**
- [ ] Website verified in Search Console
- [ ] Owner or Full user access

**Google Ads**
- [ ] Access to the Ads account
- [ ] Admin permissions
- [ ] Active campaigns exist

**Meta Ads**
- [ ] Access to Ad Account
- [ ] Admin or Analyst role
- [ ] Active ad campaigns

**Facebook Page**
- [ ] Admin access to the page
- [ ] Business page (not personal)

**Instagram**
- [ ] Business Account (not Personal)
- [ ] Connected to Facebook Business Page
- [ ] Admin access

**LinkedIn**
- [ ] Admin/Analyst role on Company Page
- [ ] Company page exists

**YouTube**
- [ ] Manager or Owner role
- [ ] At least 1 video

## Common Issues & Fixes

| Issue | Fix |
|-------|-----|
| "Access Denied" | Check you have admin permissions in that platform |
| "Property Not Found" | Verify property/page/account exists and you own it |
| No data showing | Wait 1-2 min, then refresh page |
| Data stuck loading | Clear browser cache, sign out, sign back in |
| Wrong account | Disconnect & reconnect with correct account |
| Can't find channel | Make sure it's enabled in "Connected Channels" |
